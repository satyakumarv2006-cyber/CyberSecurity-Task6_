# Cyber Security Internship - Task 6

## Task Title

Create a Strong Password and Evaluate Its Strength

## Objective

The objective of this task is to understand the importance of password security and evaluate password strength using an online password strength checker.

## Tools Used

* PasswordMeter (https://www.passwordmeter.com)
* Web Browser
* GitHub

## Files Included

* password.txt
* report.txt
* Screenshots of password strength results
* README.md

## Key Concepts Learned

* Password Strength
* Authentication
* Brute Force Attack
* Dictionary Attack
* Multi-Factor Authentication (MFA)
* Password Management
* Cyber Security Best Practices

## Observations

* Longer passwords are generally more secure.
* A combination of uppercase letters, lowercase letters, numbers, and special characters increases password strength.
* Common and predictable passwords are vulnerable to attacks.
* Strong passwords significantly reduce the risk of unauthorized access.

## Outcome

Successfully created and evaluated passwords with different complexity levels. Learned how password length, complexity, and uniqueness contribute to overall account security and protection against cyber attacks.
